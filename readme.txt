Параметры вводятся по шаблону key:value. Пример input:blalba.txt
Параметры могу вводится в любой последовательности. 
Ввод параметров с неизвесным ключем или не по формату значение вызывает ошибку!

input - обязательный параметр.Путь к входному файлу. Формат String
output - не обязательный парметр. Путь к Файлу для записи результата. Формат String
classNum - обязательный параметр. Количестов классов. Формат Int
accuracy обязательный параметр. Точность . Формат Float
startWithRandomCenters - не обязательный парметр. Начать с выбора слуачйных центров? Дефолтное состояние False". Формат Bool
columSeparator - не обязательный парметр. Разделитель CSV столбцов. Дефолтное состояние ",". Формат String
distance - не обязательный парметр. Выбор функции растояния.  Дефолтное состояние hammingDistance .  Формат hammingDistance или evclideDistance 
withoutFirstRow - не обязательный парметр. Без первой строки? Дефолтное состояние False". Формат Bool
withoutFirstColumn  - не обязательный парметр. Без первого стобца? Дефолтное состояние "False.  Формат Bool
withoutLastColumn  - не обязательный парметр. Без последнего стобца? Дефолтное состояние False". Формат Bool
